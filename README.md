# Ecommerce Database Design

This repository contains the SQL scripts for setting up a relational database for an ecommerce application.

## Database Design

- **ProductCategories**: Categories for products.
- **Users**: Users of the application.
- **Products**: Products available for purchase.
- **Orders**: Orders placed by users.
- **OrderItems**: Items within each order.

## SQL Scripts

- **create_database.sql**: Script to create the database and its tables.
- **insert_data.sql**: Script to insert sample data into the tables.
- **queries.sql**: Sample queries to fetch order statistics and delete orders.

## Submission Requirements

- Proper database design following normalization rules.
- Efficient, optimized SQL queries.
- Understanding of relational databases and indexing.

## Usage

Run the SQL scripts in your MySQL Workbench or any SQL client to set up the database.
